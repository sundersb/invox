Настоящая программа предназначена для помощи операторам АРМ ЛПУ "Релакс" при занесении подробностей о случаях онкологических заболеваний.

Причина установки: обновление к АРМ ЛПУ от 1.08.2018

Как пользоваться
****************

У программы две вкладки: первая (C00) позволяет определить код "онкологических" характеристик по указанной врачом клинической классификации опухоли (раздел II.1 инструкции к обновлению: "Онкологическое заболевание"). Вторая вкладка (Z03.1) позволяет определить нужный код направления при подозрении на онкопроцесс (раздел инструкции I "Подозрение на злокачественное новообразование").

1. C00
******

Исходя из того, что предложенные в обновлении справочники охватывают не все клинически возможные комбинации кодов, в программу добавлен интеллектуальный выбор подходящего кода. Даже если введенные Вами коды классификации в справочнике в таком сочетании не встречается, программа все равно предложит наиболее подходящий вариант.

Введите в поле "Поиск" данные, которые указал врач на статталоне (статкарте): код МКБ, клиническую группу (0-IV с возможным буквенным индексом) и стадии по классификации TNM. Всего должно получиться пять полей, разделенных пробелами. При этом букву C в коде заболевания можно не указывать, т.к. этот раздел МКБ подразумевается. Так же можно не указывать буквы T, N, M в классификации TNM. Вместо римских цифр клинической стадии можно (и удобнее) использовать обычные арабские. Например:

C20.x, стадия IIIa, T2 N1 M1 можно занести по-разному:

	"C20.3 IIIa tn N1 m1"
	"C20 3a t2 n1 m1"

Проще и удобнее всего:

	"20 3a 2 1 1"

Требования к формату перечислены выше. Главное - пять полей, разделенных пробелами. Буквы должны быть латинскими (но по умолчанию программа при включении переключает раскладку в латиницу автоматически).


Введя нужные сведения, нажмите кнопку поиска в командной панели программы (либо клавишу F2 на клавиатуре). программа интерпретирует введенные Вами данные и попытается найти то, что лучше всего подойдет для ввода. В таблице появятся возможные варианты, отсортированные по степени их уместности, первая строка автоматически выделится, и над таблицей появится код, который требуется ввести в поле МЭС1 в АРМ ЛПУ. Первый символ в коде - "x" - для того, чтобы Вы не забыли указать "повод обращения".

Если предложенный программой вариант Вас не устраивает, выберите другую строку в таблице - код для ввода обновится.

При необходимости дополните код при вводе "кодом обращения" по инструкции.


2. Z03.1
********

Поле "травма" в Релаксе никак не подходит для целей, которые на него возложили. В нем невозможно ни найти нужный элемент, ни прочитать. Возможно эта программа хоть как-то облегчит поиск кода направления, чтобы не мучить Эксель понапрасну.

Введите строку запроса в поле "Фильтр" на вкладке "Z03.1" и нажмите кнопку поиска (или F2 на клавиатуре). Выберите подходящую строку в таблице. Скопируйте код. Все.

Компьютерную томографию помогает найти строка "КТ". МРТ похоже ФОМС нам не предлагает. ПЭТ - выбор невелик, но чем богаты...

При поиске регистр символов не учитывается, все можно вводить строчными буквами.
